Desenvolvendo layouts responsivos com Bootstrap

O Bootstrap oferece um excelente modelo de grid responsivo, que facilita muito o desenvolvimento de web sites. Vejamos então como utilizá-lo.

Introdução:

Para criar um layout responsivo utilizando o Bootstrap é necessário apenas aplicar as classes do grid nos elementos HTML.


Vejamos então como fazer:

A classe que você deve utilizar é a classe col, com ela será definido alguns parâmetros (Na verdade são diversas classes já escritas mas para ficar mais fácil assimilar explicarei assim):


A classe consiste em: col-tamanho-número de colunas.

Exemplo:

col-xs-4

Que determina que ao ser exibida em um smartphone o elemento que possui esta classe será exibido em quatro colunas no grid.

Obs.: Lembrando que o grid responsivo é construido com base em 12 colunas, e no Bootstrap não é diferente, logo a soma do número de colunas na linha tem que ser igual a 12.


Esta classe podem ser aplicada em quatro tamanhos:

  • xs (Para Smartphones)
  • sm (Para Tablets)
  • md (Para Desktops)
  • lg (Para Desktops maiores)

Ou seja, para criar a responvidade você precisa apenas determinar em qual dispositivo ele irá apresentá determinado número de colunas. Simples assim, mas vejamos na prática.

Praticando:

Se em um determinado elemento, (utilizarei div como referência) se você aplicar a classe da seguinte maneira:

<div class="col-xs-12 col-sm-6"></div>

Assim o elemento em um smartphone ocupará todo espaço da linha, mas em um tablet ele ocupará apenas medade do espaço da linha ou seja seis colunas.

Vejamos em execução:

Obs.: Para visualisar o efeito redimensione a janela do seu navegador.


Mais um exemplo:

<ul>

  • <li class="col-xs-12 col-sm-4 col-md-3"> </li>
  • <li class="col-xs-12 col-sm-4 col-md-3"> </li>
  • <li class="col-xs-12 col-sm-4 col-md-6"> </li>
</ul>
